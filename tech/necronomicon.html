The Story

Necronomicon nearly ended up in the scrap heap, after its previous owner deemed it "beyond repair". The symptoms? A mouse cursor permanently stuck moving towards the top-left corner.

Determined to not let an otherwise perfectly competent machine go to waste, we did a tiny bit of poking and prodding. The only sign of physical damage to the device was a TrackPoint without the rubber cover. So we removed the keyboard, unplugged the TrackPoint ribbon cable, and the laptop was perfectly functional immediately thereafter.

It then came home with us, given a name - Blake. After upgrading the spinning rust drive to a SATA SSD, and using it as a personal Linux laptop for some time, it is now used to host a Minecraft server and our Seance proxies, and given the new name Necronomicon to fit in with the lineup.
